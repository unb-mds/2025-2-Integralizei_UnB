## Resumo das Entregas

O projeto **Integralizei UnB** atingiu seu core funcional, entregando uma solução completa para análise de vida acadêmica. As entregas foram divididas em sprints e culminaram na Release Final com as seguintes funcionalidades operacionais:

### ✅ Funcionalidades Entregues
1. **Dashboard do Aluno:** Visualização de métricas (IRA, MP, créditos) geradas a partir do PDF.
2. **Upload e Parser Inteligente:** Leitura automática de históricos do SIGAA.
3. **Calculadora de Fluxo e IRA:** Simulação de matérias futuras e impacto na integralização.
4. **Autenticação Híbrida:** Login via E-mail/Senha e Google OAuth.
5. **UnBot:** Chatbot funcional para dúvidas acadêmicas.
6. **Pesquisa de Turmas:** Mecanismo de busca e comparativo histórico.

## Release Notes

* **v1.0.0 (Release Final):** Integração total Front-end + Back-end, banco de dados PostgreSQL em produção, sistema de segurança de upload e deploy automatizado.

## Validação do Escopo

O escopo planejado nas fases iniciais foi atingido com sucesso. A transição de "teoria para prática", identificada como risco na Semana 0, foi superada. Funcionalidades complexas como a lógica de integralização e a refatoração do banco de dados para suportar escalabilidade foram concluídas nas Sprints 7 e 8. O produto final atende ao objetivo de negócio de ser uma ferramenta central para planejamento de matrícula em disciplinas.
